# RAG Feature

Purpose: retrieval, navigation maps, structured database access, vector search, and source-grounded ordinance lookup.

Subfeatures:

- `Database`: relational document schema builders and database manager.
- `VectorStore`: Chroma/Qdrant vector builders and managers.
- `Navigation`: structured navigation tools and source-map builders.

Start with `rag_run.py` for high-level retrieval/navigation entry points.
